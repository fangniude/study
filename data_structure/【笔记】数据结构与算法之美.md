#### 1. 复杂度分析

- 什么是复杂度分析？
  - 数据结构和算法，是解决 更快 和 更省的问题
  - 从 时间 和 空间 两个维度来计估
  - 分别用 时间复杂度 和 空间复杂度 来描述性能特征
  - 复杂度描述的是 算法 执行时间 或 占用空间 与 数据规模的增长关系

- 时间复杂度
  - 大O表示法
    - T(n) = O(f(n))
    - T(n)：代码执行的时间
    - n：数据规模的大小
    - f(n)：每行代码执行的次数总和
    - 大O表示法：代码执行时间随数据规模增长的变化趋势，也叫 渐近时间复杂度，简称时间复杂度
  - 复杂度分析法则
    - 单段代码看高频：比如循环、递归
    - 多段代码取最大：比如 一段代码中，有单循环 和 多重循环，取多 重循环
    - 嵌套代码求乘积：比如 多重循环、递归
    - 加法法则：总复杂度 等于 量级最大的那段代码的复杂度
    - 乘法法则：嵌套代码的复杂度等于嵌套内外代码复杂度的乘积
  - 常见时间复杂度实例分析
    - 多项式量级
      - 常量阶 O(1)
        -  不存在循环、递归
      - 对数阶 O(log n)
      - 线性阶 O(n)
      - 线性对数阶 O(n log n)
        - 归并排序、快速排序
      - 平方阶 O(n^2)、立方阶 O(n^3)、K次 阶 O(n^k)
    - 非多项式量级（NP问题）
      - 指数阶 O(2^n)
      - 阶乘阶 O(n!)
- 空间复杂度
  - 概念
    - 表示算法的存储空间 与 数据规模之间的增长关系
  - 常见的空间复杂度
    - O(1)
    - O(n)
    - O(n^2)

# 练习题

- 数组

  - 实现一个大小固定的有序数组，支持动态增删改操作
  - 实现两个有序数组合并为一个有序数组
- 链表
  - 实现单链表、循环链表、双向链表、支持增删操作
  - 实现单链表反转
  - 实现2个有序的链表合并为一个有序的链表
  - 实现求链表的中间结点

- 栈
  - 用数组实现一个顺序栈
  - 用链表实现一个链式栈
  - 编程模拟实现一个浏览器的前进、后退功能
- 队列
  - 用数组实现一个顺序队列
  - 用链表实现一个链式队列
  - 实现一个循环队列
- 递归

  - 编程实现求阶乘n！
  - 编程实现一组数据集合的全排列
- 排序
  - 实现归并排序、快速排序、插入排序、冒泡排序、选择排序

- 二分查找
  - 实现一个有序数组的二分查找算法
  - 实现模糊二分查找算法（比如大于等于给定值的第1个元素）
- 散列表

  - 实现一个LRU缓存淘汰算法
- 字符串

- 二叉树
  - 实现一个二叉查找树，并且支持插入、删除、查找操作
  - 实现查找二叉查找树中某个节点的后继、前驱节点
  - 实现二叉树前、中、后序以及按层遍历
- 堆
  - 实现一个小顶堆、大顶堆、优先级队列
  - 实现堆排序
  - 利用优先级队列合并K个有序数组
  - 求一组动态数据集合的最大Top K
- 图
  - 实现有向图、无向图、有权图、无权图的邻接矩阵和邻接表 表示方法
  - 实现图的深度优先搜索、广度优先搜索
  - 实现Dijkstra算法、A*算法
  - 实现拓扑排序的Kahn算法、DFS算法
- 回溯
  - 利用回溯算法求解八皇后问题
  - 利用回溯算法求解0-1背包问题
- 分治

- 动态规划

  - 最小路径和
